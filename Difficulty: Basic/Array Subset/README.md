<strong>Explanation: </strong>b[] is a subset of a[]</span></pre>
<pre><span style="font-size: 18px;"><strong>Input</strong>: a[] = [1, 2, 3, 4, 4, 5, 6], b[] = [1, 2, 4]

<strong>Explanation: </strong>b[] is a subset of a[]</span></pre>
